# R_Assignment
This README.md will help to know the details of the files present in the repository. 

## Data Processing

**maize_files:**
Sorted maize data with increasing or decreasing chromosome position values for the SNPs.

**teosinte_files:**
Sorted teosinte data with increasing or decreasing chromosome  position values for the SNPs

## Data Visualization

**SNP_Visualization_Shamima:**
In this file, p0 shows the total number of SNPs per chromosome, p1 shows the distribution of SNP positions across chromosomes, p2 shows SNP counts for maize only, and p3 shows SNP counts for teosinte only. 

**Homozygous_Visualization_Shamima:**
In this file, p4 shows the overall number of homozygous, heterozygous, and missing data in the entire dataset. p5 shows the proportion of those states for each sample, and p6 shows the proportion for each biological group

**My_Visualization_Shamima:**
This plot focuses only on SNPs labeled as candidate and shows where they are distributed across chromosome positions.
